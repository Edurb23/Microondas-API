# Microondas Digital — API RESTful

Este projeto faz parte do teste técnico e entrega os níveis 3 e 4 do desafio, atendendo os requisitos de:

- Cadastro e listagem de programas de aquecimento pré-definidos e customizados
- Persistência de dados em JSON
- Autenticação de usuário (cadastro e login)
- Proteção de endpoints com JWT Bearer Token
